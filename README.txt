/**   使用说明： **/

 1，没有sql文件，因为里面也没什么有营养的东西，都是自己随便发的

 2，安装的时候，需要修改两个地方：一、找到db.php 文件修改里面的主机名称和密码，注意：是主机的名称！ 二、找到install.php文件，同样修改主机名和密码。跟修改db.php一样。

 3，然后运行install.php文件，如果你是直接放在域名下，那就是：http://www.你的域名/install.php ，如果是文件里那就是：http://www.你的域名/文件名/install.php

 4,自动安装。最后注册一下自己的主账号。 

 5，登陆：/admin.php

 6,发文章，发心情吧。

7，里面除了index.php之外还有个index1.php是不同的首页风格，看个人喜好，如果要切换的话，重命名一下就行了，把喜欢的风格改成index.php就好。
